Function
---
- Login
  - remember ID
  - auto login
  - register

- Enrollment
  - look up subjects
  - add selected subjects to cart
  - apply for selected subjects
  - remove subjects in cart or application
  - move subjects in cart to application or vice versa
  - save enrollment data by id
